# Sociality 

An social media app dealing with :
  - Firebase as backend.
  - Bloc(Cubit) as state management.
  - real time data
  - FCM notification


# Featuers

  - Authentication (login - register).
  - (Create-Delete) post.
  - Followers & Following.
  - (Add - Delete) Friend request.
  - Get Friend requests.
  - (Get - Update) profile.
  - Get following posts.
  - (Add - Delete) likes.
  - (Add - Delete) comments.
  - search.
  - chat.
